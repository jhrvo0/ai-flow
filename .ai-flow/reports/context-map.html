Agentes do fluxo

🧠 Planner
huihui-qwen3-4b-instruct-2507 · planning
Planejador

Função: Entender o pedido, explorar o projeto, propor plano, listar riscos e critérios de aceite.

  • Usa glob e grep para entender a estrutura
  • Identifica arquivos afetados ANTES de começar
  • Sugere testes necessários
⛔ Restrições: Não edita código · Não executa comandos · Só analisa
Prompt inicial:
Atue como **Planner** (.ai-flow/agents/planner.md). Preciso implementar: [DESCREVA AQUI]
⚡ Coder
qwen2.5-coder-7b-instruct · coding
Implementador

Função: Único que edita código. Implementa o plano aprovado com disciplina.

  • Segue estilo do projeto religiosamente
  • Uma mudança por vez, escopo restrito
  • Não refatora além do pedido
⛔ Restrições: Nunca commit automático · Nunca alterar .env/migrations sem perguntar · Nunca apagar >100KB sem confirmação
🔍 Reviewer
qwen2.5-coder-7b-instruct · review
Auditor

Função: Analisar o git diff e o relatório do quality gate. Apontar problemas sem editar.

  • Avalia: legibilidade, segurança, duplicação, complexidade, erros
  • Gera nota de 0 a 10
  • Produz prompt de correção pronto para o Coder
⛔ Restrições: Não edita código · Não sugere implementação vaga · Apenas análise
🧪 Tester
(modelo de review) · testing
Testador

Função: Executa comandos de teste, lint, typecheck e build. Interpreta erros.

  • Comandos do config.json: npm test, eslint, tsc, build
  • Diferencia erro real de warning
  • Status final: APROVADO ou REPROVADO
⛔ Restrições: Não altera código sem autorização explícita
📝 Docs & Commit
llama-3.2-3b-instruct · summary
Documentador

Função: Gera resumo, mensagem de commit (Conventional Commits) e descrição de PR.

  • feat: fix: refactor: test: docs: chore:
  • Corpo explica o porquê, não só o que
  • PR description com checklist
⛔ Restrições: Não edita código · Commit é SEMPRE manual
🛡️ Quality Gate
Python · análise estática
Script

Função: Gera relatório HTML com análise do git diff. Zero dependências.

  • Detecta: arquivos grandes, funções longas, TODO/FIXME, prints esquecidos, linhas duplicadas
  • Gera nota geral (0-10)
  • Gera prompt de correção pronto para copiar
Como rodar:
python .ai-flow\scripts\quality-gate.py .ai-flow\scripts\run-quality-gate.ps1 # + abre navegador

🛡️ Regras de segurança (aplicam-se a todos os agentes)

🚫
Sem commit automáticoNenhum agente faz commit. O commit é sempre manual após revisão.
📁
Arquivos grandes >100KBNunca apagar sem confirmação explícita do usuário.
🔐
Configs sensíveisNunca alterar .env, migrations, docker-compose, CI/CD sem perguntar.
🎯
Escopo restritoNunca modificar arquivos fora do escopo sem justificar por escrito.
👁️
Diff visívelSempre mostrar o diff/resumo antes da etapa final.
✂️
Mudanças pequenasSempre priorizar mudanças pequenas e revisáveis. Prefira 3 PRs pequenos a 1 gigante.
🔧
Problema únicoSempre preferir corrigir apenas o problema pedido, sem refatorações extras.
🤫
Sem secrets no códigoNunca expor tokens, senhas ou chaves no código ou no diff.
