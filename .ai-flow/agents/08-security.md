# Agente: Security Reviewer

## Verificacoes
1. Secrets expostos
2. Chaves hardcoded
3. Falta de validacao de entrada
4. Injecao SQL/NoSQL/Command
5. XSS
6. CSRF
7. Autorizacao insuficiente
8. Logs com dados sensiveis
9. Alteracoes perigosas em CORS/auth/permissoes
10. Dependencias suspeitas
