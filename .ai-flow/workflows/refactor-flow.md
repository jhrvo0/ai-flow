# Workflow: Refactor Flow

1. Orchestrator avalia risco tecnico.
2. Context Engineer mapeia areas impactadas.
3. Planner divide em fases pequenas.
4. Architect valida acoplamento e fronteiras.
5. Aprovacao humana por fase.
6. Coder + Patch Applier executam fase.
7. Reviewer, Security e Tester validam.
8. Docs/Commit + Memory atualizam decisoes.
